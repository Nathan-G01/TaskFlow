# Create A Basic Pong Game In Python

Goal:
Produce a small, playable Pong game in Python inside `sample/workspace/pong.py`.

Constraints:
- Keep the implementation simple and easy to run locally.
- Prefer the Python standard library first.
- If graphics are needed, keep dependencies minimal and explain them in code comments or output.
- Do not modify repository structure outside `sample/workspace/` unless required for the task.

Definition of done:
- `sample/workspace/pong.py` exists.
- The game is playable locally by running the Python file.
- Two paddles can be controlled from the keyboard.
- The ball moves, bounces on walls, and interacts with paddles.
- A visible score is maintained.

Execution guidance:
- Work in small, bounded steps.
- Validate each step before proposing the next one.
- Prefer incremental progress over large one-shot rewrites.
